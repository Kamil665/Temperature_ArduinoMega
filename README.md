# 🌡️ Система контроля температуры на Arduino Mega

Проект представляет собой систему мониторинга температуры на базе **Arduino Mega**, использующую датчик **DS18B20**, ЖК-дисплей **LCD 16x2**, сервопривод, кнопку управления, вентилятор и светодиодную индикацию.

---

# Возможности

- 🌡️ Измерение температуры с помощью DS18B20
- 📟 Отображение информации на LCD 16x2
- 🔄 Управление сервоприводом
- 🌀 Управление вентилятором
- 💡 Индикация состояния светодиодами
- 🔘 Управление режимами при помощи кнопки
- 💻 Вывод информации в Serial Monitor

---

# Используемые компоненты

- Arduino Mega 2560
- DS18B20
- Резистор 4.7 кОм
- LCD 16x2
- Потенциометр 10 кОм (для контрастности LCD)
- Сервопривод SG90
- Кнопка
- Светодиод
- Красный светодиод
- Вентилятор (или его имитация светодиодом)
- Макетная плата
- Провода

# Подключение DS18B20

```
DS18B20          Arduino Mega

GND  ----------> GND

VCC  ----------> 5V

DATA ----------> A5
                 |
                 |
             4.7 kΩ
                 |
                 |
                5V
```

Без резистора 4.7 кОм датчик чаще всего работать не будет.

# Подключение кнопки

```
D2 -------- Кнопка -------- GND
```

Используется режим

```cpp
INPUT_PULLUP
```

поэтому внешний резистор не требуется.

---

# Подключение вентилятора

Если используется настоящий вентилятор:

- рекомендуется подключать через транзистор;
- напрямую к Arduino подключать нельзя.

Если вентилятора нет, можно подключить обычный светодиод к пину D3.

---

# Работа программы

После запуска выполняются следующие действия:

1. Инициализация всех устройств.
2. Поиск датчика температуры.
3. Чтение температуры.
4. Вывод температуры в Serial Monitor.
5. Проверка подключения датчика.
6. При отсутствии датчика выводится сообщение об ошибке.

---

# Вывод в Serial Monitor

При исправном подключении:

```
Device count: 1
Temperature: 24.75
```

Если датчик не найден:

```
Device count: 0
Error: Sensor disconnected or wiring issue!
```

---

# Возможные проблемы

## Device count: 0

Причины:

- отсутствует резистор 4.7 кОм;
- перепутаны провода;
- неправильный пин;
- неисправный датчик.

---

## Temperature: -127.00

Это означает, что библиотека не смогла получить данные с датчика.

Проверьте:

- питание;
- GND;
- DATA;
- резистор;
- качество соединений.
